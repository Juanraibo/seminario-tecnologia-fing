## 📐 Parámetros Recomendados por Herramienta

### Midjourney (Discord)
```
Sufijos recomendados:
--ar 16:9          (aspect ratio para presentaciones)
--ar 4:3           (aspect ratio para productos)
--ar 1:1           (aspect ratio cuadrado para iconos)
--style raw        (más realista, menos artístico)
--v 6              (versión 6 de Midjourney)
--q 2              (calidad máxima)

Ejemplo completo:
/imagine A professional photo of a discarded LCD monitor --ar 4:3 --style raw --v 6 --q 2
```

---

### DALL-E 3 (ChatGPT Plus)
```
Prefijo recomendado:
"I NEED to test how the tool works with extremely simple prompts. DO NOT add any detail, just use it AS-IS:"

Luego el prompt simple. Esto evita que DALL-E "mejore" tu prompt.

Ejemplo:
I NEED to test how the tool works with extremely simple prompts. DO NOT add any detail, just use it AS-IS: A professional photo of a used laptop computer, electronic waste, white background, studio lighting
```

## 🎯 Tips Generales para Mejores Resultados

### Para Fotos de Productos (RAEE)
1. **Siempre incluir:** "white background, studio lighting, product photography"
2. **Para realismo:** "professional photo, 8K, high detail, realistic"
3. **Evitar:** términos artísticos como "beautiful", "artistic", "creative"

### Para Interfaces UI
1. **Especificar:** "clean minimal design, modern UI, dark mode"
2. **Colores exactos:** incluir códigos hex cuando sea crítico
3. **Referencias:** "Tailwind CSS style", "SaaS dashboard aesthetic"

### Para Infografías
1. **Simplicidad:** "minimal design, clean lines, flat design"
2. **Colores corporativos:** siempre mencionar emerald green y blue
3. **Estilo:** "vector style, professional infographic"

### Para Contexto/Escenarios
1. **Iluminación:** especificar "natural lighting" o "studio lighting"
2. **Ambiente:** describir el espacio (university, industrial, office)
3. **Realismo:** "professional photography, realistic scene"
