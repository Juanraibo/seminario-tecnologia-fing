## 🛠️ Herramientas Recomendadas

### Para Fotos Realistas de Productos (RAEE)
**Mejor opción:** Midjourney v6  
**Alternativa:** Leonardo.ai (Photo Real mode)  
**Gratis:** Stable Diffusion XL (local o Hugging Face)

**Por qué:** Necesitas realismo para clasificación con IA

---

### Para Mockups UI
**Mejor opción:** Tomar screenshots reales del sistema  
**Alternativa:** Midjourney + Figma para retoques  
**Rápido:** Canva con shapes y componentes

**Por qué:** Screenshots reales son más creíbles

---

### Para Infografías y Diagramas
**Mejor opción:** Canva (templates + customización)  
**Alternativa:** Figma (desde cero, más control)  
**Con IA:** DALL-E 3 + edición en Canva

**Por qué:** Más fácil iterar y ajustar texto

---

### Para Presentación Final
**Recomendado:** Canva + Screenshots reales  
**Premium:** Canva Pro (más templates)  
**Gratis:** Google Slides + screenshots reales

**Por qué:** Balance tiempo/calidad/colaboración

## 📐 Formatos Recomendados

### Para Presentación:
- **Aspect Ratio:** 16:9 (1920x1080px o 1280x720px)
- **Screenshots:** PNG (mejor calidad) o JPG (menor peso)
- **Diagramas:** SVG o PNG de alta resolución

### Para Web:
- **Imágenes RAEE:** 800x600px (JPG, optimizadas)
- **Logos:** SVG (escalable) o PNG transparente
- **Banners:** 1920x600px (hero) o según necesidad

### Para Documentación:
- **Capturas:** PNG con compresión media
- **Diagramas:** SVG exportado como PNG para Markdown

## 💡 Tips Universales

### Para Cualquier Prompt de IA:
1. **Ser específico** con colores (incluir hex codes)
2. **Incluir "professional"** para evitar estilo cartoon
3. **Especificar background** (white, dark, transparent)
4. **Agregar "clean minimal design"** para estética moderna
5. **Iterar 3-5 veces** hasta lograr calidad deseada

### Para Screenshots:
1. **Datos realistas** antes de capturar (no placeholders)
2. **Zoom 100%** en el navegador
3. **Limpiar UI** de elementos temporales (toasts, errors)
4. **Modo oscuro** para consistencia
5. **Full size capture** en Chrome DevTools

### Para Canva:
1. **Usar grilla** (View → Show Rulers and Guides)
2. **Alinear todo** (Select All → Position → Align Center)
3. **Espaciado consistente** (múltiplos de 8px)
4. **Exportar HD** (Share → Download → PNG → Recommended quality)

## 🔄 Workflow Completo — Opción Recomendada

### Día 1 (1 hora) — Screenshots del Sistema
```bash
cd app
npm run dev
```

1. Abrir http://localhost:5173

## 📞 ¿Necesitas Ayuda?

### Si un prompt no funciona:
1. **Simplificar:** Quitar adjetivos y detalles
2. **Cambiar herramienta:** Probar en otra IA
3. **Iterar:** Cambiar un parámetro a la vez

### Si la calidad no es buena:
1. **Aumentar resolución:** Pedir "8K" o "high detail"
2. **Agregar "professional":** Evita estilos amateur
3. **Especificar estilo:** "product photography", "studio lighting"

### Si necesitas algo que no está en los prompts:
1. **Combinar prompts:** Tomar partes de 2-3 prompts
2. **Describir con detalle:** Ser específico sobre lo que quieres
3. **Incluir referencia:** "in the style of [ejemplo conocido]"
