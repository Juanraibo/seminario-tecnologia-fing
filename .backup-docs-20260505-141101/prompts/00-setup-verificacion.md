# Prompt 00 — Verificación del Setup
# Usar al inicio de la Sesión 4, antes de desarrollar

---

Leé el archivo `CLAUDE.md` en la raíz del proyecto para entender el contexto completo.

Luego verificá que la estructura del proyecto esté correcta:

1. Confirmá que existen estos archivos y que su contenido es válido:
   - `app/package.json`
   - `app/src/main.jsx`
   - `app/src/App.jsx`
   - `app/src/context/AppContext.jsx`
   - `app/src/data/lotes.json` (debe tener al menos 5 lotes en distintos estados)
   - `app/src/constants/estados.js`

2. Verificá que `app/src/App.jsx` tenga rutas definidas para:
   - `/login`
   - `/instituto`
   - `/ecopunto`
   - `/gestora`
   - `/admin`
   - `/trazabilidad`

3. Si algún archivo falta o está incompleto, crealo o completalo siguiendo las convenciones del `CLAUDE.md`.

4. Haceme un resumen de lo que encontraste y si el proyecto está listo para empezar el desarrollo.
